# 🐛 Scripts de Depuración

Esta carpeta contiene scripts de auditoría, verificación e inspección usados durante el desarrollo.

## ⚠️ Importante

**Estos archivos NO se rastrean en Git** (ver `.gitignore`).

Son herramientas de desarrollo para:
- Auditar calidad de datos
- Verificar integridad de pipelines
- Inspeccionar features y estructuras
- Comparar datasets

## 📝 Convenciones

- `auditoria_*.py`: Scripts de auditoría de datos
- `verificar_*.py`: Scripts de verificación de integridad
- `inspect_*.py`: Scripts de inspección de estructuras
- `comparar_*.py`: Scripts de comparación de datasets
- `check_*.py`: Scripts de chequeo rápido

## 🧹 Mantenimiento

Revisar periódicamente y documentar hallazgos importantes en:
- Issues de GitHub
- Documentación del proyecto
- Notebooks de análisis

**Luego eliminar scripts obsoletos.**
